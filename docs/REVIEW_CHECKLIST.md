## G. 对“可以继续推进”的判定

只有下面这些条件都满足，才建议进入下一阶段：

* [ ] 当前阶段目标已真实生效
* [ ] `.venv` 测试通过
* [ ] 真实样例没有明显阻塞问题
* [ ] 没有越界实现
* [ ] 输出结果已足够作为下一阶段输入

如果上面任何一项不满足，应优先建议：

* 小范围修复
* 规则修正
* 或重新验证

而不是直接推进。
